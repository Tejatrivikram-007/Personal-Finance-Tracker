## User Guide
To use the Personal Finance Tracker, follow these steps:
1. **Registration**: Create an account by filling out the registration form. Ensure you provide a valid username and password.
2. **Login**: After registration, log in using your credentials to access your dashboard.
3. **Adding Records**: Navigate to the respective sections (Income, Expenses, Savings) to add new records. Fill out the forms and submit.
4. **Viewing Statements**: Use the statements feature to generate financial reports for specific months and years. Choose the desired format (CSV, PDF, Excel) for download.

## Installation Instructions
To set up the Personal Finance Tracker locally, follow these steps:
1. Clone the repository from GitHub.
2. Navigate to the project directory.
3. Install the required dependencies using `pip install -r requirements.txt`.
4. Run the application using `python manage.py runserver`.
5. Access the application in your web browser at `http://127.0.0.1:8000`.
